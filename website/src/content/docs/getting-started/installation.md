---
title: Installation
description: Install Forgeplan on your system
---

## Using Cargo (recommended)

```bash
cargo install forgeplan
```

## Using Homebrew

```bash
brew install forgeplan
```

## Using Shell Script

```bash
curl -fsSL https://forgeplan.dev/install.sh | sh
```

## Verify Installation

```bash
forgeplan --version
forgeplan health
```
